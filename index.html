Welcome to FPMCAE v1.0

FPMCAE is a general-purpose simulation analysis software for complex structural behaviors, developed by Professor Luo Yaozhi's team at Zhejiang University. It uses the Finite Particle Method as its core algorithm and integrates pre-processing and post-processing functions, various solver engines, and parallel computing technology.

Software Features

  • Core Algorithm: Finite Particle Method for efficient simulation analysis of complex structural behaviors.
  • Pre- and Post-Processing: Powerful tools for data input and result analysis.
  • Solver Engines: Supports various types of solvers for different simulation needs.
  • Parallel Computing: Integrated technology to enhance computational efficiency, suitable for large-scale simulations.

Application Areas

  • Civil Engineering and Transportation
  • Mechanical Manufacturing
  • Aerospace
  • Other fields requiring complex structural behavior simulation

欢迎使用 FPMCAE v1.0

FPMCAE 是由浙江大学罗尧治教授团队自主研发的通用结构复杂行为仿真分析软件。采用有限质点法作为核心算法，集成了前后处理功能和多种求解器引擎，并融入了并行计算技术。

软件功能

  • 核心算法：有限质点法，高效进行结构复杂行为的仿真分析。
  • 前后处理功能：强大的数据输入和结果分析工具。
  • 多种求解器引擎：支持多种类型的求解器，满足不同仿真需求。
  • 并行计算：集成技术提升计算效率，适合大规模模拟。

应用领域

  • 土木交通
  • 机械制造
  • 航空航天
  • 其他需要结构复杂行为仿真分析的领域
